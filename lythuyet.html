Phần Lý thuyết (40 điểm)

1. position: absolute và position: relative (10 điểm)

So sánh

  • position: relative
    • Thuộc normal flow (không bị loại khỏi layout)
    • Dịch chuyển dựa trên vị trí ban đầu
    • Vẫn giữ khoảng không gian ban đầu
  • position: absolute
    • Out of flow (bị loại khỏi layout)
    • Định vị theo phần tử cha gần nhất có position ≠ static
    • Nếu không có → theo body

Ví dụ thực tế


.card {
  position: relative;
}

.badge {
  position: absolute;
  top: 0;
  right: 0;
}

→ Dùng để đặt badge "Sale" ở góc sản phẩm

Responsive & Deploy

  • Dùng absolute với px cố định → dễ vỡ layout trên mobile
  • Nên dùng %, flexbox, grid để tương thích đa thiết bị

2. Domain, Hosting và DNS (10 điểm)

Quy trình

  1. Mua domain (example.com)
  2. Thuê hosting (server lưu code)
  3. Upload code HTML/CSS/JS
  4. Cấu hình DNS trỏ domain → hosting

DNS là gì?

DNS (Domain Name System) chuyển domain thành IP


example.com → 192.168.1.1

Công cụ deploy

  • GitHub Pages
  • Netlify

Rủi ro bảo mật

  • Không dùng HTTPS → dễ bị tấn công
  • Lộ API key trong JavaScript
  • Cấu hình DNS sai → bị redirect sang web giả

3. let, const, var (10 điểm)

So sánh

  • var: function scope, hoisting → undefined
  • let: block scope, có TDZ
  • const: block scope, không gán lại

Ví dụ lỗi


for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 100);
}
// output: 3 3 3
      

for (let i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 100);
}
// output: 0 1 2

Giải thích

var dùng chung 1 biến i → nên in ra 3 lần số 3.
let tạo block scope → mỗi vòng lặp là 1 giá trị riêng.

Debug

  • Mở DevTools (F12)
  • Console.log
  • Breakpoint
  • Pause on exception

4. Event Delegation (10 điểm)

Khái niệm

Event Delegation là kỹ thuật gắn event vào phần tử cha, sử dụng event bubbling để xử lý sự kiện từ phần tử con.

Ví dụ


document.getElementById("list").addEventListener("click", function(e) {
  if (e.target.tagName === "LI") {
    alert(e.target.textContent);
  }
});

Ứng dụng

Danh sách động (render bằng JavaScript)

Lợi ích

  • Giảm số lượng event listener
  • Tăng hiệu suất
  • Hỗ trợ phần tử thêm động
